## VI. 页面类型规范

### 1. 封面页 (01_cover.svg)

- 背景：纯白主背景 + 底部浅科技灰装饰带

- 顶部：华为红横向装饰条 + 科技灰双线分隔

- 装饰元素：左上角多层同心圆（抽象华为logo），右侧边缘垂直科技装饰线

- 标题区：居中/左对齐布局，搭配华为红分隔线与几何矩形装饰

- 品牌区：左上角/右下角预留华为官方logo标准放置位，符合品牌安全间距规范

### 2. 目录页 (02_toc.svg)

- 页眉：固定「目录 / CONTENTS」标题，搭配左侧华为红竖标

- 列表布局：左右双栏无衬底卡片列表布局，保证信息清晰高效

- 设计规范：采用「华为红大编号 + 标题 + 灰色下划线」组合，适配性强，层级清晰

- 装饰元素：左侧垂直装饰线 + 竖向「Index」辅助文字，平衡视觉

### 3. 章节过渡页 (02_chapter.svg)

- 背景：全屏深科技灰(`#191919`)或华为红(`#E60012`)背景，强化章节区隔

- 视觉中心：左侧华为红垂直标识条 + 白色标题组合，视觉焦点明确

- 右侧装饰：极简科技线条阶梯效果 + 斜向点阵纹理，营造科技纵深

- 背景元素：巨型半透明章节编号，低透明度处理，不干扰主内容

### 4. 正文内容页 (03_content.svg)

- 布局：全开放无框布局，去除中心边框，最大化内容展示空间，适配数据图表、方案介绍等多场景

- 核心信息：顶部区域采用极简左侧华为红线 + 核心结论文本呈现，降低视觉干扰

- 边界处理：四角仅保留极淡的「安全区域锚点」，无冗余边框

- 页脚规范：包含数据来源、页码、章节名称，保持品牌一致性

### 5. 结束页 (04_ending.svg)

- 首尾呼应：布局与封面页高度一致，形成完整的闭环视觉体验

- 核心内容：居中致谢语，华为品牌logo放置在醒目位置

- 联系信息：底部宽幅联系信息卡片，极简无冗余设计

- 装饰元素：左右对称多层华为红同心圆，与封面设计语言统一

## VII. 推荐布局模式

### 1. 核心结论单页布局

- 顶部核心信息栏呈现单句核心结论/价值主张

- 下方搭配单张大图、数据图表或重点强调文本，最大化信息传递效率

- 适用于核心观点输出、关键数据展示场景

### 2. 卡片网格布局

- 白色内容区内放置2×2或3×2数据/方案卡片

- 推荐卡片底色：`#F5F5F5`(浅科技灰)，重点卡片使用`#FFF0F1`(浅红)，圆角统一4px

- 适用于多模块数据、多维度能力展示场景

### 3. 分栏对比布局

- 左侧呈现现状/痛点/问题，右侧呈现华为解决方案/成果/价值

- 中间可使用科技箭头作为逻辑连接符

- 适用于前后对比、问题-方案、竞品对比场景

### 4. 左图右文/左文右图布局

- 一侧放置产品图、架构图、场景图，另一侧放置核心文本描述

- 图文占比推荐6:4或7:3，保证视觉平衡

- 适用于产品展示、解决方案介绍、架构说明场景

## IX. SVG技术约束规范

### 强制规则

1. viewBox固定为 `0 0 1280 720`，禁止修改

2. 必须包含全屏`<rect>`作为背景层，覆盖整个画布

3. 文本换行必须通过`<tspan>`实现，禁止自动换行

4. 透明度必须使用`fill-opacity` / `stroke-opacity`单独设置，禁止群组统一设置

5. 箭头必须使用`<polygon>`实现，禁止使用`marker`属性

6. 所有颜色必须使用十六进制HEX格式，禁止使用`rgba()`/`rgb()`格式

### 禁用元素黑名单

- `clipPath`、`mask` 裁剪/遮罩元素

- `<style>`、`class` 外部样式表（`<defs>`内的`id`允许使用）

- `foreignObject` 外部嵌入对象

- `textPath` 路径文本

- `animate`、`animateTransform`、`set` 动画元素

- `<g opacity="...">` 群组统一透明度（透明度必须在单个元素上单独设置）

## XI. 使用注意事项

1. **Logo适配规范**：推荐使用华为官方标准logo，深色背景使用白色反白logo，浅色背景使用标准红色logo，严格遵守logo安全间距规范（周边留白不小于logo高度的1.5倍）。

2. **字体安装说明**：推荐优先安装「HarmonyOS Sans」鸿蒙黑体（华为官方免费商用字体），以实现最佳品牌视觉效果；无该字体环境下，自动适配微软雅黑兜底字体。

3. **扩展色使用规则**：如需新增扩展色，必须保持华为红+科技灰的核心主调不变，辅助色仅限小范围点缀使用，禁止破坏品牌统一风格。

4. **视觉一致性原则**：所有装饰元素必须遵循极简科技风格，避免过度装饰，始终以信息传递的清晰性为第一优先级。

5. **品牌合规要求**：所有应用场景需遵守华为全球品牌规范手册，禁止修改logo形状、比例、颜色，禁止违规使用品牌元素。
