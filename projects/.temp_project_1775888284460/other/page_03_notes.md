# 第 3 页：整体业绩核心特征

## 演讲备注

**四大核心特征**:

1. **增长质量双优**
   - 营收与净利润增速均超 65%
   - 毛利率持续上行至 71.1%
   - 费用增速（+45%）远低于营收增速（+65.5%）
   - 运营效率大幅优化

2. **结构高度集中**
   - 数据中心业务占总营收 89.7%
   - 成为绝对核心增长引擎
   - 其他业务协同增长

3. **现金流超级充沛**
   - 经营现金流破 1027 亿美元
   - 自由现金流近 970 亿美元
   - 现金储备超 625 亿美元
   - 具备极强战略扩张与抗风险能力

4. **增长韧性强劲**
   - 全年四个季度均实现环比正增长
   - 无季度下滑记录
   - AI 需求从爆发式增长转向稳健持续增长
